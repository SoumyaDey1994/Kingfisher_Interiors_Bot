4.3.6. Smart Insert Material
This option helps the user to avoid searching a Material or Color from the Library when the Material or
Color name (or a part of it) is known. The limitation of Smart Insert Material is that it will not be getting
results from the Palette. It searches the entire Library which might be time consuming.
